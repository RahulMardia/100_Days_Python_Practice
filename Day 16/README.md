# Day 16 - Coffee Machine (Object-Oriented Programming)

This folder contains my implementation of the **Coffee Machine Project**, built as part of **Day 16** of the **100 Days of Python Bootcamp**.

## Project Overview

### Coffee Machine

A command-line coffee machine simulator that allows users to purchase different types of coffee while managing available resources and payments.

The machine supports:

- Espresso
- Latte
- Cappuccino

The program checks:

- Resource availability
- Coin payments
- Transaction success
- Remaining ingredients after each order

## Topics Covered

- Object-Oriented Programming (OOP)
- Classes and Objects
- Creating and Importing Modules
- Methods and Attributes
- Working with Multiple Classes
- Encapsulation
- Code Organization and Reusability

## Features

- Resource tracking
- Coin processing
- Profit tracking
- Inventory management
- Object-oriented design
- Multiple coffee selections
