# Day 14 - Higher Lower Game

This folder contains my implementation of the **Higher Lower Game**, built as part of **Day 14** of the **100 Days of Python Bootcamp**.

## Project Overview

### Higher Lower Game

A command-line game where the player must guess which person, brand, or organization has more social media followers.

The game presents two options:

- Option A
- Option B

The player must predict which option has a higher follower count. For each correct answer, the score increases and the game continues. The game ends when an incorrect choice is made.

## Topics Covered

- Functions
- Dictionaries
- Loops
- Randomization
- Comparing data from dictionaries
- Game logic and state management
- Modularizing code using separate files

## Features

- Random selection of accounts
- Score tracking
- Continuous gameplay until a wrong answer
- Dynamic comparison between entities
- Console screen clearing between rounds
