## Project Included

### Calculator
A command-line calculator that performs basic arithmetic operations and allows users to continue calculations using previous results.

The program supports:

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)

The program:

1. Accepts two numbers from the user.
2. Lets the user choose an operation.
3. Calculates and displays the result.
4. Allows the user to continue with the current result or start a new calculation.
