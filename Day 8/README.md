## Project Included

### Caesar Cipher
A Python program that encrypts and decrypts messages using the Caesar Cipher technique.

The program:

1. Asks whether the user wants to `encode` or `decode` a message.
2. Accepts a text message.
3. Accepts a shift number.
4. Shifts each letter by the specified amount.
5. Displays the encrypted or decrypted result.
6. Allows the user to restart the program.
