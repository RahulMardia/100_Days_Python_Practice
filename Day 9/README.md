## Project Included

### Secret Auction
A command-line program that collects bids from multiple participants and determines the highest bidder.

The program:

1. Asks each participant for their name.
2. Records their bid amount.
3. Stores all bids in a dictionary.
4. Continues until there are no more bidders.
5. Finds and announces the highest bidder.
