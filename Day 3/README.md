## Project Included

### Treasure Island
A simple interactive adventure game where the player makes choices to find hidden treasure.

The game asks the user to make decisions such as:

1. Choosing between left or right
2. Waiting or swimming
3. Selecting the correct colored door

Depending on the choices, the player either wins the treasure or loses the game.
