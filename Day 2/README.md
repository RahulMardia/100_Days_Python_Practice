## Project Included

### Tip Calculator
A Python program that calculates how much each person should pay when splitting a bill, including the selected tip percentage.

The program takes:

1. Total bill amount
2. Tip percentage (10%, 12%, or 15%)
3. Number of people splitting the bill

It then calculates and displays the amount each person should pay.
