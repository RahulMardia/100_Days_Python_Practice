# Day 12 - Scope and Number Guessing Game

This folder contains my solutions and practice exercises for **Day 12** of the **100 Days of Python Bootcamp**.

## Project Overview

### Number Guessing Game

A command-line game where the computer randomly chooses a number between 1 and 100, and the player must guess it within a limited number of attempts.

The game includes:

- Random number generation
- Difficulty levels (Easy and Hard)
- Attempt tracking
- Feedback for guesses that are too high or too low
- Win and lose conditions

## Topics Covered

- Local Scope
- Global Scope
- Constants and Global Variables
- Variable Visibility
- Using Functions to Organize Code
- Debugging Logic
- Building a Complete Guessing Game
